# 모바일 지뢰찾기 앱

## 게임 설명
* 사각형은 가로 9줄, 세로 9줄
* 지뢰는 총 10개이며 위치는 랜덤
* 자신을 제외한 주변 8개 사각형에 포함된 지뢰의 개수를 각 사각형에 표시
* 하단 토글 버튼으로 깃발 표시 / 타일 부수기 전환 가능
* 남은 지뢰의 개수가 상단에 표시됨
* 모든 비지뢰 타일을 부수거나 모든 지뢰의 위치를 깃발로 표시하면 게임 승리
* 지뢰를 클릭하면 게임 패배
* 게임 승리 혹은 패배 후 재시작 가능

## 개발 환경
* JAVA 1.8
* Android Studio 2022.1.1
